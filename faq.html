Frequently Asked Questions (FAQ)

Q. What is The Steam Curator?

The Steam Curator (tSC) is a curated game store. This means that all games present on the store have undergone a review process. The mission for tSC is to improve visibility for steam games that don't reach the audience they deserve.

Q. Does The Steam Curator have regional pricing?

The Steam Curator does not provide regional pricing for games and currently, only support USD($). This is due to hardware/cost limitations, but we would love to implement regional pricing in the future.

Q. How can I get a game on The Steam Curator?

If you're an enthusiastic fan and think a game deserves to be on this platform you can enter an application over on our discord server. If you are a developer you can also use our discord but use the developer application instead.

Q. How are games selected?

Games are reviewed individually and have to pass a series of tests to make it onto the storefront. The point of the review system is to avoid asset-flip and low effort games that have increased in numbers on the steam store. This will hopefully increase visibility for talented indie developers.

The review process will be based on several factors including:

  • Steam reviews:
    A game will require to have at least or greater than 40% average review rating on steam. Review bombing will be taken into account in the context of the situation. Games which have a less than 40% average steam rating will need to provide reasoning as to why their rating is at that level.
  • Reproducibility:
    Reproducibility determines how much effort was put into a game. A high reproducibility could me two things. The game uses stolen or existing assets or the game is small in scale. Game which are small in scale will not be affected by this metric.
  • Uniqueness:
    This metric will be used in conjunction with the reproducibility metric. It determines how unique the game is. If the game is easily repoducible and is not unique than it is most likely an asset flip type game which are unlikely to appear on this store front.
  • Popularity:
    If the game is popular it will have a higher chance of passing the review process.

Games do not need to fullfill all the requirements but have a higher chance if they do

Q. Why does The Steam Curator need donations?

The Steam Curator requires donations for server costs. If donations are not able to cover server costs than an alternative monitisation system will have to be implemented (i.e Ads).

Q. How is steam data collected?

The data for steam games in obtained through steams unofficial API. Alternative images for steam games are obtained using steam grid DB API. More information about these API can be found on https://wiki.teamfortress.com/wiki/User:RJackson/StorefrontAPI and https://www.steamgriddb.com/api/v2

Q. Does The Steam Curator collect or distribute user data?

The Steam Curator does not collect any user data.
